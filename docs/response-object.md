# Object Response

To convert the response of DataTables to an object, just pass `true` on `make` api.

```php
use DataTables;

Route::get('user-data', function() {
	$model = App\User::query();

	return DataTables::eloquent($model)
				->addColumn('intro', 'Hi {{$name}}!')
				->toJson();
});
```
